# FULiveDemo-Swift

FULiveDemo-Swift 是 iOS 平台上，集成相芯人脸跟踪及视频特效开发包（简 Nama SDK）的Swift版本的集成示例。

目前集成了 Faceunity 面部跟踪、美颜功能。
